# Package index

## Fetching Data

Functions for downloading enrollment data from RIDE

- [`fetch_enr()`](https://almartin82.github.io/rischooldata/reference/fetch_enr.md)
  : Fetch Rhode Island enrollment data
- [`fetch_enr_multi()`](https://almartin82.github.io/rischooldata/reference/fetch_enr_multi.md)
  : Fetch enrollment data for multiple years
- [`get_available_years()`](https://almartin82.github.io/rischooldata/reference/get_available_years.md)
  : Get available years for Rhode Island enrollment data

## Data Transformation

Functions for transforming and tidying data

- [`tidy_enr()`](https://almartin82.github.io/rischooldata/reference/tidy_enr.md)
  : Tidy enrollment data
- [`id_enr_aggs()`](https://almartin82.github.io/rischooldata/reference/id_enr_aggs.md)
  : Identify enrollment aggregation levels
- [`enr_grade_aggs()`](https://almartin82.github.io/rischooldata/reference/enr_grade_aggs.md)
  : Custom Enrollment Grade Level Aggregates

## Caching

Functions for managing local data cache

- [`cache_status()`](https://almartin82.github.io/rischooldata/reference/cache_status.md)
  : Show cache status
- [`clear_cache()`](https://almartin82.github.io/rischooldata/reference/clear_cache.md)
  : Clear the rischooldata cache
