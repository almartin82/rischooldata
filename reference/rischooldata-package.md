# rischooldata: Fetch and Process Rhode Island School Data

Downloads and processes school data from the Rhode Island Department of
Education (RIDE). Provides functions for fetching enrollment data from
RIDE's Data Center and transforming it into tidy format for analysis.
Supports October 1st Public School Student Headcounts from 2010-2011 to
present.
